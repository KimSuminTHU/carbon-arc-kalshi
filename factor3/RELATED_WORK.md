## 요약 — 우리 연구의 위치
선행연구는 **(A) 금융텍스트→성과**, **(B) LLM 금융텍스트→성과**, **(C) 카드 alt-data→매출/실적**을 각각 입증했다. 그러나 **카드 거래 alt-data(X) + 어닝콜 텍스트(Z)를 LLM으로 결합해 매출 서프라이즈를 예측하고, X-only/Z-only 대비 보완성을 post-training-cutoff OOS에서 검정**한 연구는 확인되지 않는다(§E). 또한 최근 **LLM look-ahead 누설** 문헌(§D)이 우리의 "cutoff 이후만 평가" 설계를 정확히 정당화한다.

## C. Alt-data → 매출/실적 nowcasting
| 출처 | 방법 | 타깃 | 발견 |
|---|---|---|---|
| **Fleder & Shah, "Forecasting with Alternative Data" (2019)** ✓ (ACM 10.1145/3366694; ScienceDaily 2019) | 익명 주간 카드거래 + 분기 실적 모델 | 기업 재무/실적 | **Wall St 컨센서스를 분기의 57.2%에서 격파.** 카드→실적 nowcast의 학술 근거 |
| 실무(Consumer Edge / Earnest / Facteus / Second Measure) ✓ (deweydata, facteus 가이드) | 머천트→티커 매핑, 소비 트렌드 | 매출/earnings surprise | 소비지출이 **최대 3분기 앞** earnings surprise를 양(+)으로 예측 |

해석: **X-only(카드→매출) 자체가 컨센서스를 modest하게 이김** — 우리 repo 결과(r≈0.19)와 일치. 단 이들은 **텍스트 미결합**.

## D. 보완성·점증 예측력 방법론 + LLM 누설 통제
**점증/보완성 검정:**
- **Nested 점증 R²** + Fama-MacBeth t (cross-section).
- **Diebold & Mariano (1995)** 예측정확도 비교; **Harvey, Leybourne & Newbold (1997)**, *Int. J. Forecasting* 13(2):281–91 ✓ — 소표본(<20) t(n−1) 보정. (우리 `dm_hln` 재사용; n≈105라 보정 중요.)
- **Clark & West (2007)**, *J. Econometrics* 138(1):291–311 ✓ — **nested OOS** 전용 보정(DM은 nested에서 큰 모델을 부당하게 페널티 → MSPE-adjusted). **X ⊂ X+Z 비교의 정석.**
- **Permutation / shuffle surrogate**, **BH-FDR**(다중검정) — repo `surrogate()`·`cluster_boot()` + FDR.

**LLM look-ahead 누설 (우리 설계의 핵심 근거):**
- **Liang, "Look-Ahead Bias in Financial Forecasts Generated by LLMs"** ✓ (SSRN 6772819) / **Noguer i Alonso** ✓ (SSRN 5022165) / **"AI's predictable memory in financial analysis"**, *Economics Letters* ✓ (S0165176525004392): GPT-4(cutoff 2021-09-30) 기준 **pre-cutoff 타깃의 절대오차가 16–18% 낮음** = 메모리제이션으로 백테스트 성능이 부풀려짐.
- **완화책(문헌 권고): (i) 학습 cutoff 이후 표본으로 제한, (ii) time-stamped LLM, (iii) 식별정보 제거.** → 우리는 **(i)을 채택**(테스트 = report > 2025-12-01).
- 관련: **DatedGPT**(arXiv 2603.11838) 등 time-aware 사전학습.

**정보 vs 거래성:** rev-surprise→earnings-day return은 약함(우리 repo ≈0.07) → nowcasting 주장과 trading 주장을 분리.

## E. Novelty & Gap
세 줄기(A/B/C)는 각각 성숙했으나 **교집합이 비어 있다:**
- **Kim-Muhn-Nikolaev**: LLM이 재무제표→이익방향, **alt-data·매출서프라이즈 아님**.
- **Jha et al**: LLM이 콜→capex 스코어, **카드 결합·매출서프라이즈 아님**(단 Option A 방법론 선례).
- **ACL 2023 / FinCall-Surprise**: 텍스트(±음성/이미지)→earnings surprise, **거래/카드 데이터 결합 없음**.
- **QLoRA-LLM (2408.06634)** (WebFetch 확인): 재무지표+어닝콜+시장지수+애널리스트등급→주가방향, **카드 alt-data 없음, X/Z 보완성 ablation 없음**.
- **Fleder-Shah / 실무**: 카드→실적, **텍스트 미결합**.

**⇒ 우리 기여(novel):** *카드 거래 alt-data(X) + 전분기 어닝콜(Z)을 LLM으로 결합 → 매출 서프라이즈를, X-only/Z-only 대비 **보완성**을 명시 검정하고, **post-cutoff OOS**로 누설을 통제한 최초 류의 연구.* (간단하지만 빈 칸 — Jacob의 "simple해도 아무도 안했으면 OK"에 부합.)

## F. 우리 설계에 반영 (방법론 채택)
1. **필수 baseline**(문헌 표준): naive·**애널리스트 컨센서스**(N0), **LM 사전/FinBERT**(N2) — "LLM이 값싼 NLP를 이기는가" 답(Huang 2023 식 비교).
2. **LLM→해석가능 스코어**(Option A) = Jha식; 0–100 anchored + multi-prompt 평균.
3. **누설**: 학습 cutoff 이후만 평가(look-ahead 문헌 권고) — 우리의 105 event.
4. **보완성 검정**: Clark-West(nested OOS) + DM-HLN + shuffle surrogate + BH-FDR.
5. **정보 vs 거래성** 분리 명시.
